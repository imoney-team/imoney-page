iMoney

事件

var i = 0;
//绑定命名空间
$result.on("touchstart.name", function() {
    i++;
    if (i >= 2) {
        //基于命名空间移除事件
        $result.off("name");
    }
    console.log("iMoney.js");
});
//绑定事件
$result.on("touchend", function() {
    //移除事件
    $result.off("touchend");

模块化

import 'imoney/src/module';

配置

$.config(object),object有4个可选参数如下：

参数 描述 默认值
waitSeconds 设置认为模块加载失败的秒数 7
baseUrl 基础路径 './'
paths 映射那些不直接放置于baseUrl下的模块名 {}
urlArgs url参数，例如时间戳、查询参数

例如：

$.config({
    //基础路径
    baseUrl:'module/'
})

定义模块

例如：新建module.js，module.js依赖run.js，则

//module.js
$.define(["run"],function(run) {
    console.log("module");
    return {
        color: "red",
        size: "unisize"
    }
});

使用模块

例如：依赖module.js，

$.require(["module"],function(module){
    console.log(module.color+'----'+module.size)
})
